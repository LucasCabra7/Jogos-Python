-> 💯 Sobre o projeto:

Elaboração de jogos utilizando Python3 no editor Pycharm. Inicialmente há uma página inicial que pergunta ao jogador quais jogos ele deseja jogar, o encaminhando para o jogo desejado, em seguida o jogo é iniciado. Com o intuito de elaborar algo produtivo, será perguntado ao jogador qual nível de dificuldade ele deseja se aventurar.
